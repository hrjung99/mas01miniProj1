4,5일차

   12. 게시물 관리
       1) 게시물 목록
         1) DB에 등록된 모든 게시물을 출력한다 
         2) 로그인 한 경우 등록 버튼이 활성화 된다 
         3) 로그인 하지 않았을 경우 등록버튼을 비활성화 또는 화면에 버튼이 출력되지 않는다 
         4) 검색 기능으로 제목 또는 내용을 검색하여 출력할 수 있게 한다 
         
      2) 게시물 상세보기 
         1) view_count 1씩 자동 증가 
         2) 로그인 한 사람이 작성자 또는 관리자이면 삭제, 수정 버튼 출력
         
      3) 게시물 삭제 (ajax -> fetch () 함수로 기능 구현)
         1) 로그인 한 사람이 작성자 또는 관리자이면 삭제 가능함 
         2) 해당 권한이 없으면 작업 취소 
         
      4) 게시물 수정 (양식은 페이지 이동, 처리는 ajax -> fetch () 함수로 기능 구현)
         1) 로그인 한 사람이 작성자 또는 관리자이면 수정 가능함 
         2) 해당 권한이 없으면 작업 취소 
         
       5) 게시물 등록 (양식은 페이지 이동, 처리는 ajax -> fetch () 함수로 기능 구현)
         1) 로그인 한 경우만 게시물 등록 가능 
         2) 로그인 사람 이름 입력하지 않고 자동으로 로그인 한 사람 자동등록 할 수 있게 구현한다
